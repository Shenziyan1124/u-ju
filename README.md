# u-ju

U Ju 活动平台是一个活动报名管理系统，包含管理端、服务端和移动端相关模块，适合用于活动发布、活动报名、用户管理和后台运营管理场景。

## 项目模块

```text
.
├── u-ju-admin/    # 管理端
├── u-ju-server/   # 服务端
├── u-ju-uniapp/   # 移动端 / uni-app
├── docs/          # 项目文档
└── PRD-v1.0.md    # 产品需求文档
```

## 项目特点

- 活动报名管理系统
- 前后端分离结构
- 包含产品需求文档和项目文档
- 支持管理端、服务端、移动端协同开发

## 使用说明

分别进入对应子目录，根据各模块内的依赖配置安装并启动项目。
